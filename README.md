# Students Randomizer
[Available Here](https://francoisdoussin.github.io/students-randomizer/)
## Requirements

- nodejs
- npm/yarn

## Install

```shell
npm install
npm start
```

## Use

Load a csv file with at least `fistname` and `lastname` keys.
Example :

```csv
firstname, lastname
Grace, Hopper
Ada, Lovelace
```
